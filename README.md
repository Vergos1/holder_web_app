## About

Holder is a beauty services booking application currently in early development. The project is built with a modern stack — Next.js App Router, Shadcn UI components, TanStack Query for server state, Zustand for client state and Better Auth for authentication.

> ⚠️ This project is in early development and not publicly available yet.

## Features

- **Authentication** — Better Auth for user sessions
- **Server state** — TanStack Query for data fetching and caching
- **Client state** — Zustand for local state management
- **Shadcn UI + Radix UI** — accessible component system
- **Axios** — HTTP client for API requests
- **Husky + Commitlint** — conventional commits enforcement
- **Import sorting** — ESLint plugin for auto-sorted imports

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
